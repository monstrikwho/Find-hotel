# Поиск отелей (тестовое задание)

### React приложение, в котором реалзиована фильтрация постов по трем критериям
- кол-во звезд отеля
- кол-во отзывов
- цена

### Приложение построено на
- React
- Redux, Redux-thunk
- MUI
